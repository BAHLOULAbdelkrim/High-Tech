High-Tech Hugo site (generated)

Instructions:
1. Download and unzip this archive into a folder.
2. Move the folder to your Hugo sites directory or open a terminal in the folder.
3. Ensure you have Hugo installed (extended version not required for this setup).
4. Run: hugo server
5. Open http://localhost:1313/

Notes:
- This site uses a minimal Tailwind CDN via <script src="https://cdn.tailwindcss.com"></script> in the head.
- The theme is located in themes/hightech.
- Slider interval is set to 5000 ms (5 seconds) via config.toml param sliderInterval.
